# Frontend Styling Standard

## 目标

- 保持 `antd` 为主要 UI 基础。
- 样式尽量局部化，避免跨页面污染。
- 页面视觉服务于聊天和后台管理工作流，不做营销页式装饰。

## 当前约定

- 通用组件优先使用 `antd`。
- 图标优先使用 `lucide-react`。
- 页面样式跟随页面目录，例如 `pages/Chat/ChatPage.css`。
- 全局样式只放真正全局的 reset、字体和基础变量。

## 布局原则

- 后台页面优先信息密度、可扫描性和稳定布局。
- 聊天页面优先输入、消息阅读和历史会话切换。
- 按钮和工具入口优先使用清晰图标与 tooltip。
- 固定格式组件应有稳定尺寸，避免内容变化造成布局跳动。
